# Implementation Example — Der Gott des Gemetzels

This folder is a **scenario implementation**, not a reusable template layer.

It is designed so the player can begin as either:
- Penelope Longstreet
- Michael Longstreet

The guest couple is modeled here as:
- Annette Reille
- Alain Reille

This keeps the scenario outside the generic templates while still showing how IDs, parent links, and inject stacks work in practice.
